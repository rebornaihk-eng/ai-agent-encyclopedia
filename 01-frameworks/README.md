# AI Agent Frameworks

This section covers open source frameworks for building AI agents.

---

## Categories

### Multi-Agent Frameworks
- **CrewAI** - Role-based team orchestration
- **AutoGen** - Microsoft's multi-agent conversation framework
- **LangGraph** - State machine-based agent workflows

### General Purpose
- **LangChain** - Most popular, versatile framework
- **Semantic Kernel** - Microsoft's enterprise SDK
- **Haystack** - NLP-focused with agent capabilities

### Specialized
- **Phidata** - Data-focused agents
- **Agents** - Lightweight agent framework
- **OpenAI Swarm** - Experimental multi-agent orchestration

---
