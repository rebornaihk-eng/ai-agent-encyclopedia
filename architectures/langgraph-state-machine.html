🔀 LangGraph State Machine

📚 Source: LangChain Official Docs, GitHub (26K stars)

LangGraph is a low-level orchestration framework for building, managing, and deploying long-running, stateful agents as graphs.

📐 Official Architecture

Source: LangGraph GitHub Repository

🧩 Key Concepts

StateGraph

The core building block that defines nodes, edges, and state management.

State (TypedDict)

A shared state object passed between nodes. Each node can read and modify it.

from typing_extensions import TypedDict

class State(TypedDict):
    messages: list[str]
    count: int
    context: str

Nodes

Functions that process state and return updates.

def node_a(state: State) -> dict:
    # Process state
    return {"messages": state["messages"] + ["Hello"]}

Edges

Define flow between nodes. Can be conditional.

from langgraph.graph import START, END

graph = StateGraph(State)
graph.add_node("node_a", node_a)
graph.add_edge(START, "node_a")
graph.add_edge("node_a", END)

📝 Complete Example

from langgraph.graph import START, END, StateGraph
from typing_extensions import TypedDict

# 1. Define State
class State(TypedDict):
    text: str
    count: int

# 2. Define Nodes
def node_a(state: State) -> dict:
    return {"text": state["text"] + "a", "count": state["count"] + 1}

def node_b(state: State) -> dict:
    return {"text": state["text"] + "b"}

def should_continue(state: State) -> str:
    if state["count"] < 3:
        return "loop"
    return "end"

# 3. Build Graph
graph = StateGraph(State)
graph.add_node("node_a", node_a)
graph.add_node("node_b", node_b)

# Add edges
graph.add_edge(START, "node_a")
graph.add_conditional_edges(
    "node_a",
    should_continue,
    {"loop": "node_b", "end": END}
)
graph.add_edge("node_b", "node_a")

# 4. Compile and Run
app = graph.compile()
result = app.invoke({"text": "", "count": 0})
print(result)  # {'text': 'ababab', 'count': 3}
