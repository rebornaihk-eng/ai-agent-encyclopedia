# AI Agent Encyclopedia 🔐

A comprehensive, Wikipedia-style resource covering AI agents, frameworks, coding tools, design patterns, and learning resources.

## Features

- **21 Chapters** across 6 sections
- **50+ Project Ideas** with difficulty levels
- **Architecture Diagrams** with official sources
- **Code Templates** for LangChain, CrewAI, LangGraph
- **Password Protected** for private access

## Access

**Password:** `ai-agent-2025`

Change it in `index.html` line 94.

## Contents

| Section | Description |
|---------|-------------|
| **Reference** | SOTA models, frameworks, coding agents |
| **Design** | 10 patterns + 6 architecture diagrams |
| **Innovate** | 50+ project ideas + success stories |
| **Build** | Code examples + deployment templates |
| **Learn** | Learning path + resources + cost calculator |
