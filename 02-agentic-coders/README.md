# Agentic Coders

AI-powered coding assistants that can write, edit, and debug code autonomously.

---

## Categories

### IDE-Integrated
- **Cursor** - VS Code fork with AI-first approach
- **GitHub Copilot** - Most popular, integrates everywhere
- **Windsurf** - Codeium's AI-native IDE
- **Void** - Open source alternative

### CLI-Based
- **Claude Code** - Anthropic's official CLI
- **Aider** - Open source terminal-based
- **OpenCode** - Open source alternative

### Extension-Based
- **Continue.dev** - Open source VS Code extension
- **Codeium** - Free alternative to Copilot
- **Tabnine** - Long-established AI completion
