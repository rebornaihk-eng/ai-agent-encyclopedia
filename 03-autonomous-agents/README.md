# Autonomous Agents

Self-directed AI agents that can plan and execute multi-step tasks.

---

## Categories

### Task Automation
- **AutoGPT** - First autonomous agent, task decomposition
- **BabyAGI** - Task management and execution
- **SuperAGI** - Agent infrastructure platform

### Code Generation
- **GPT Engineer** - Full codebase generation
- **MetaGPT** - Multi-role software company
- **Devin** - First AI software engineer (commercial)

### Research
- **GPT Researcher** - Autonomous research
- **AgentGPT** - Web-based autonomous agents
